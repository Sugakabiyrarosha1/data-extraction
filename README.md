This repository is a **demonstration of my skills in data extraction** from a variety of sources.  
It contains **practical examples, scripts, and ETL workflows** for extracting, cleaning, and preparing data from websites, documents, spreadsheets, and databases.  

---

## 📌 Overview  

The goal of this project is to **showcase efficient and reliable techniques** to extract and preprocess data from both structured and unstructured sources.  
These examples are designed to be **adaptable to real-world workflows** across different domains, from analytics to machine learning.  

---

## 🔍 Key Features  

- **🌐 Web Scraping**  
  Extract data from static and dynamic websites using:  
  - BeautifulSoup  
  - Selenium  
  - Scrapy  
  - Playwright (for headless automation)  

- **📑 Document Parsing**  
  Retrieve data from common file formats:  
  - PDFs (`PyPDF2`, `pdfplumber`)  
  - Word documents (`python-docx`)  
  - Text/HTML files  

- **📊 Spreadsheet Handling**  
  Work with:  
  - Excel (`openpyxl`, `pandas`)  
  - CSV  

- **🗄️ Database Extraction**  
  Connect to and extract data from relational databases:  
  - Mssql  
  - MySQL / PostgreSQL (`SQLAlchemy`, `psycopg2`)  
  - 

- **☁️ Big Data Pipelines **  
  Examples with:  
  - Hadoop (HDFS)  
  - Hive & Pig  
  - MongoDB integration
